# Item Decoration

## Item Decoration 的作用
简单理解可以为在 Item 周围绘制一圈的内容. 细化一下来说有两点:
1. 指令 Item 的 Offset
2. 在 Offset 的区域里绘制图案

很容易想到一个很常见的应用就是添加一条分割线了.
